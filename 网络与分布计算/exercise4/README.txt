先运行bean.WebService发布服务，然后运行client.Client订阅端口，进入时输入用户名和密码进行注册或登录。
运行过程中每条命令执行后都会有命令提示，所以不提供help命令。

使用与测试流程---------->

注册与登录：
qiyu 1039

开始时间晚于结束时间：
add 2020-10-16/20:00 2020-10-15/19:00 title1
正确输入：
add 2020-10-16/20:00 2020-10-16/21:00 title1
时间冲突：
add 2020-10-16/20:00 2020-10-16/22:00 title2
正确输入：
add 2020-10-16/22:00 2020-10-16/23:00 title2

正常排序命令 输出（title1 title2）：
query 2020-10-16/01:00 2020-10-16/23:30

错误时间格式：
add 2020-10-17/08:00 2020-1017/10:00 title3
正确输入：
add 2020-10-17/08:00 2020-10-17/10:00 title3

正常排序命令：
query 2020-10-16/01:00 2020-10-17/23:30
输出（title1 title2 title3）

正确删除命令：
delete 2
查看删除成功：
query 2020-10-16/01:00 2020-10-17/23:30
输出（title1 title3）

重复删除 报错：
delete 2

正确清空命令：
clear
查看清空结果 输出（None item !）：
query 2020-10-16/01:00 2020-10-17/23:30

退出命令：
quit

重登使用错误密码 qiyu 1000 报错
重登使用不同的用户名 直接注册
